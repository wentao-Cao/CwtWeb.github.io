Linux资源配置管理

Posted on 2020-06-07 Edited on 2020-06-08 In Linux笔记
阿里云大学Linux基础——资源查询的基本操作
#### 进程信息 ps -ef ps -ef|gre mysql ps -ef|gre ssh ps -ax 查看进程的列表 ps-aux 显示进程以及所有者 cd ssh/ 服务 sshd_config 配置文件 服务器远程连接端口 /etc/init.d 放置命令文件

进程状态

top 显示所有进程的 输入：1 显示多个CPU 显示-内存-交换区- clt +c 退出
xbc 进程的排序-查看

系统进程数目

pstree 以树展开

杀掉进程

kill -1 pid 重跑
kill -9 pid 杀死当前进程
kill -15 pid 结束进程 pid 表示进程ID

其他

free -m 查看内存使用
lspci 查看所有PCI总线设备
df -h 查看分区使用情况
在 .sh等操作 运行文件后加 & 会返回进程 ID 可以通过ps查询 将程序运行在后台
ctrl+z 展缓到后台 fb 继续运行 bg后台运行
查看后台进程 jobs

定时任务

crontab -l 查看定时任务
crontab -e 编辑定时任务
eg： 0 1 * * * run.sh 每天1点执行run.sh
eg: */3 * * * * tar-zcf /ect/k.txt

压缩/解压 文件

tar -czf myfile.tar /home
tar -zvxf myfile.tar.gz -C resrotr/
