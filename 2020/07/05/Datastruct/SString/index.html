数据结构_串

Posted on 2020-07-05 Edited on 2020-07-20 In 数据结构

根据考试大纲，整理串的基本知识点。

串

1）基本要求

掌握串的相关概念、串的存储结构（顺序串和链式串）及基本运算的实现；

定义：串是由零个或者多个字符组成的有限序列，又名叫字符串。

一般记为：s=“a1a2a3a4a5a6…”,长度为n,零个字符的串称为空串。

空格串：只包含空格的串，有长度而且不止一个空格。

子串与主串的区别：串中的任意个数的连续的字符组成的子序列称为该串的子串，相应的，包含子串的串称为主串。

子串在主串中的位置：就是子串的第一个字符在主串中的序号。

字符串的比较：是通过组成串的字符之间的编码来进行的，而字符的编码指的是字符在对应字符集中的序号。

比较两个字符串：1、若 a=“a s d” b =”a s d f g” ——>a<b 2：若 a = “a b c d e” b =”a b c e” 由于第四个字符e>d 所以 b>a.

串的顺序存储：用一组地址连续的存储单元来存储串中的字符序列。按照预定义的大小，为每一个定义的串变量分配一个固定长度的存储区，一般使用定长的数组来定义。一般可以将实际的串的长度保存在数组0的下标位置。或者在串尾部加上 “\0”表示结束。

串的链式存储结构：一个节点存储多个字符，最后一个节点若是未被占满的，可以用“#”或者其他字符补全。

堆分配存储表示：任是一组地址连续的存储单元存放串值字符序列，但是他们的存储空间是在程序执行过程中动态分配而得。

掌握KMP算法的基本思想及模式匹配过程；

能灵活运用串的特点解决复杂的应用问题。

2）考试范围：

串类型的定义；

串的定长顺序存储、堆分配存储、块链存储表示和实现；

串的模式匹配算法；

串的应用。
